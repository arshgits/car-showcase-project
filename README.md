# car-showcase-project
A sleek and dynamic car showcase slider built using vanilla JavaScript, delivering smooth animations and an engaging user experience. Designed with modern UI techniques and optimized performance without relying on heavy libraries.

✨ Features:

Interactive image carousel using JavaScript DOM manipulation
Smooth custom animations (contentIn / contentOut) for transitions
Modern UI with CSS gradients, transforms, and shadow effects
Optimized logic using appendChild & prepend for better performance
Integrated Font Awesome icons for enhanced visuals

💡 A lightweight and visually appealing frontend project focused on performance, creativity, and clean design.
